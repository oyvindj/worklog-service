!Build
```
.gradlew build
```
!Deploy to Cloud Foundry
```
.gradlew build;cf push
```
!Technologies

- Springboot
- Mixed Java & Groovy code in Gradle
- Mongo Database as Cloud Foundry Service
- Spring Data Repository
- Spring Security
- REST SERVICES
- JWT for authentication from client
- Deployment to Cloud Foundry as Java Buildpack
- Auth0 API integration for authentication
- Runs as separate service from frontend
